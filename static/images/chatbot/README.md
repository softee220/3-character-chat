# 🖼️ 이미지 폴더

이 폴더에 챗봇에 사용할 이미지 파일을 추가하세요.

## 📋 필수 이미지 파일

AI 응답에 따라 자동으로 표시되는 이미지 파일들입니다. 다음 6가지 이미지를 준비해야 합니다:

### 1. 공감 (empathy.png)
- **설명**: 사용자의 감정에 공감하는 모습
- **사용 시점**: "알겠어", "이해해", "같아", "맞아", "그렇구나", "공감" 등의 키워드가 포함된 응답
- **기본값**: 다른 이미지가 선택되지 않으면 이 이미지가 표시됩니다

### 2. 무조건적인 지지 (support.png)
- **설명**: 사용자를 무조건적으로 지지하고 응원하는 모습
- **사용 시점**: "응원", "힘내", "화이팅", "넌 할 수 있어", "믿어", "좋아", "멋져" 등의 키워드가 포함된 응답

### 3. 놀람 (surprise.png)
- **설명**: 놀란 모습
- **사용 시점**: "와", "헐", "진짜", "대박", "와우", "놀랐" 등의 감탄사가 포함된 응답
- **우선순위**: 가장 높음 (다른 키워드보다 우선)

### 4. 단호한 조언 (advice.png)
- **설명**: 단호하고 확고한 조언을 하는 모습
- **사용 시점**: "해야 해", "해야겠어", "필요해", "중요해", "무조건", "절대", "반드시" 등의 키워드가 포함된 응답

### 5. 웃는 모습 (laughing.png)
- **설명**: 웃거나 즐거워하는 모습
- **사용 시점**: "ㅋㅋ", "하하", "웃", "재밌", "흐흐", "ㅎㅎ" 등의 키워드나 웃는 이모지(😀😆😂🤣😊😄)가 포함된 응답

### 6. 눈치보는 모습 (careful.png)
- **설명**: 조심스럽고 눈치보는 모습
- **사용 시점**: "혹시", "괜찮아?", "불편하면", "부담", "아니면", "안 되면" 등의 키워드가 포함된 응답

## 📁 파일 구조

```
static/images/chatbot/
├── empathy.png                    # 공감 이미지
├── support.png                    # 무조건적인 지지 이미지
├── surprise.png                   # 놀람 이미지
├── advice.png                     # 단호한 조언 이미지
├── laughing.png                   # 웃는 모습 이미지
└── careful.png                    # 눈치보는 모습 이미지
```

## 🔧 이미지 선택 로직

이미지는 `services/chatbot_service.py`의 `_select_image_by_response()` 메서드에서 자동으로 선택됩니다.

**선택 우선순위**:
1. 놀람 (surprise)
2. 단호한 조언 (firm_advice)
3. 웃는 모습 (laughing)
4. 공감 (empathy)
5. 무조건적인 지지 (unconditional_support)
6. 눈치보는 모습 (careful)

**기본값**: 키워드가 매칭되지 않으면 공감(empathy) 이미지가 표시됩니다.

## 📝 참고사항

- 이미지 파일은 PNG, JPG, JPEG, GIF 형식을 지원합니다.
- 권장 크기: 300x300px 이상 (프론트엔드에서 최대 300px로 표시됨)
- 파일명은 정확히 위와 같이 지정해야 합니다 (대소문자 구분).
- 이미지가 없으면 해당 응답에서는 이미지가 표시되지 않습니다 (에러 없음).
